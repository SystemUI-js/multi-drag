## 特性（Features）

支持多个手指**同时**拖动多个元素（当然也支持鼠标拖动）

Support MULTI fingers drag MULTI Items(also support mouse dragging)

## 现代化打包构建工具，支持TS（Modern tools of Project Building, support TypeScript）

基于Vite + TypeScript打造

Build with Vite + TypeScript
